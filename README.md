# Cross The Road

A browser-based 3D crossing game built with plain HTML, CSS, JavaScript, and Three.js.

## How to run

Open `index.html` in a browser. No build step or install is required.

## Controls

- Arrow keys or WASD to move
- Tap or swipe on mobile
- P to pause
- M to mute

## Gameplay

Dodge cars, trucks, and trains, ride logs across water, and try to beat your best score.
